## About The Project

This Laravel Project is used to learn how to upload imag ein laravel. If you want to using it make sure your databse name is same with .env file, also don't forget to migrate the migration file to your database if you want to make this project work properly.

PS : 
- Laravel only can accept file bellow than 2mb
